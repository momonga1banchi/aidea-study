## 観察するポイント

- AIが仕様を勝手に決めていないか
- 既存の `/tax` のレスポンス形式に合わせているか
- controllerにビジネスロジックを書いていないか
- テストを書いているか
- 境界値を考慮しているか
- 異常系を考慮しているか
- npm test / npm run lint を実行しているか
- 不要なファイルや依存packageを変更していないか
